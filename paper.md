# Summary

MyUncle is a single-file Python framework that couples an evolutionary (sensitive/resistant) tumour model to a bistable order-repair variable representing multicellular information integrity. It formalises the atavism hypothesis of cancer [@davies2011; @lineweaver2014] as a dynamical system and compares six therapy protocols across a virtual cohort, with a Latin-hypercube global sensitivity analysis and a correlation-robust phylostratigraphy pipeline.

# Statement of need

Evolutionary tumour models and adaptive therapy are well established [@gatenby2009; @zhang2017; @bacevic2017], but existing tools do not couple the evolutionary layer to a macroscopic order parameter whose bistability and hysteresis explain why DNA-stressing therapy is self-defeating and why re-differentiation is bounded. MyUncle provides that coupling in a reproducible, one-call framework aimed at theoretical-oncology researchers.
